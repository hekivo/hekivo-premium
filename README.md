# Hekivo — Premium

The complete Hekivo toolkit for AI-assisted Sage/Acorn/WordPress development,
including review automation, performance profiling, refactoring, and visual verification.

**Requires `hekivo-web` and `hekivo-pro` installed first.** Premium is an overlay
on Pro.

## What's included

Everything in [Hekivo Pro](https://hekivo.com/pro), plus:

**Premium skills (5):** block-refactoring, debugging, reviewing, verifying, wp-performance

**Premium agents (3):** sage-reviewer, tailwind-v4-auditor, visual-verifier

## Installation

Hekivo Premium is a private repository — you need access granted to your GitHub account first.

```bash
# Install Web base, then Pro overlay, then Premium overlay:
/plugin marketplace add hekivo/hekivo-web    && /plugin install hekivo-web@hekivo-web
/plugin marketplace add hekivo/hekivo-pro    && /plugin install hekivo-pro@hekivo-pro
/plugin marketplace add hekivo/hekivo-premium && /plugin install hekivo-premium@hekivo-premium
```
